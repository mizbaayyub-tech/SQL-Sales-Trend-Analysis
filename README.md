# SQL-Sales-Trend-Analysis
# Sales Trend Analysis Using SQL

## Objective
Analyze monthly revenue and order volume using SQL aggregate functions.

## Tools Used
- SQL Server Management Studio (SSMS)

## Dataset
online_sales

## SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT
- SUM()
- COUNT(DISTINCT)
- YEAR()
- MONTH()
- GROUP BY
- ORDER BY

## Output
The query displays monthly revenue and total order volume for each month.

## Result
January 2024:
Revenue = 1200
Orders = 2

February 2024:
Revenue = 1500
Orders = 2

March 2024:
Revenue = 3000
Orders = 3
